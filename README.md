# CineAi Genesis

**Ваш интеллектуальный помощник по созданию короткометражных фильмов**

CineAi Genesis помогает режиссерам и сценаристам создавать креативные концепции для короткометражных фильмов с детальными раскадровками.

## Возможности

- **Консервативный подход** - Классические драматические и комедийные сюжеты
- **Эксперимент 1: Магический реализм** - Смешение реального и сверхъестественного  
- **Эксперимент 2: Авангард** - Радикальные подходы и нарушение правил
- ** Детальные раскадровки** - 5 ключевых кадров с полным описанием
- ** Визуальный стиль** - Цветовые палитры, референсы, операторская работа
